Tentang Kuis

Web ini dibuat untuk memenuhi tugas projek Praktek Kerja Lapangan kami. Web ini juga dibuat sebagai sarana kami dalam mempelajari teknologi-teknologi yang digunakan untuk membuat web.

Web ini dibuat dibuat dengan menggunakan beberapa teknologi seperti untuk bagian frontend kami menggunakan framework Tailwind CSS untuk styling dan vanilla JavaScript untuk interaktifitasnya. Sedangkan untuk bagian backend kami menggunakan Node.js dengan frameword Express.js dan database MySQL.

Adapun fitur-fitur yang ada pada web ini antara lain:

  • Login & Register
  • CRUD Kuis
  • Kotak Pencarian
  • Like untuk Kuis favoritmu
  • Leaderboard
